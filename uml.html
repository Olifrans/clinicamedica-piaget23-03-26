• Paciente 1..* Consulta | Médico 1..* Consulta
• Controlador → DataService (dependência)
• Controlador → Validador (dependência)
• ChatbotIA → Controlador (consulta dados)
• ReportGenerator → DataService (leitura dados)
• NotificationService → Consulta, Paciente (envio notificações)
• Interface → Controlador, ChatbotIA, ReportGenerator, NotificationService

🔄 Diagrama de Sequência - Agendamento com Notificações

Paciente → Interface: Solicita agendamento
Interface → Controlador: agendarConsulta(dados)
Controlador → Validador: validarDados(dados)
Validador → Controlador: dados válidos
Controlador → DataService: salvarConsulta(consulta)
DataService → Controlador: confirmação (id)
Controlador → NotificationService: agendarLembrete(consultaId, 1440)
NotificationService → Controlador: lembrete agendado
Controlador → Interface: atualizarView()
Interface → Paciente: exibir confirmação

🔄 Diagrama de Sequência - Chatbot IA

Paciente → ChatbotIA: "Quero agendar uma consulta"
ChatbotIA → ChatbotIA: analisarIntencao()
ChatbotIA → Controlador: consultarHorariosDisponiveis()
Controlador → DataService: getConsultas()
DataService → Controlador: listaConsultas
Controlador → ChatbotIA: horariosDisponiveis
ChatbotIA → ChatbotIA: gerarRespostaContextual()
ChatbotIA → Paciente: "Horários disponíveis: 09:00, 14:00..."
Paciente → ChatbotIA: "09:00"
ChatbotIA → Controlador: solicitarAgendamento(dados)
Controlador → DataService: salvarConsulta()
DataService → Controlador: confirmacao
Controlador → ChatbotIA: sucesso
ChatbotIA → Paciente: "Consulta agendada com sucesso!"

🔄 Diagrama de Sequência - Geração de Relatório

Administrador → Interface: Seleciona tipo de relatório
Interface → ReportGenerator: gerarRelatorio(tipo, periodo)
ReportGenerator → DataService: buscarDados(tipo, periodo)
DataService → ReportGenerator: dadosBrutos
ReportGenerator → ReportGenerator: processarDados()
ReportGenerator → ReportGenerator: gerarMetricas()
ReportGenerator → Interface: dadosProcessados
Interface → Chart.js: renderizarGrafico(dados)
Interface → Administrador: exibir relatório e gráfico
Administrador → Interface: clicar em "Exportar PDF"
Interface → html2pdf: gerarPDF(elemento)
html2pdf → Administrador: arquivo PDF baixado
