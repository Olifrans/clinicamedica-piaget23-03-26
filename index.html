    📋 Levantamento de Requisitos v2.0

    Requisitos Funcionais

    • RF01 - Cadastrar, editar e excluir pacientes
    • RF02 - Cadastrar, editar e excluir médicos
    • RF03 - Agendar, confirmar, realizar e cancelar consultas
    • RF04 - Visualizar dashboard com métricas
    • RF05 - Filtrar e buscar registros
    • RF06 - Validar CPF e datas de consulta
    • RF07 - Chatbot IA para atendimento automatizado
    • RF08 - Geração de relatórios analíticos com gráficos
    • RF09 - Sistema de notificações por email/sms simuladas
    • RF10 - Lembretes automáticos de consultas
    • RF11 - Exportação de dados em PDF e Excel
    • RF12 - Dashboard com indicadores inteligentes

    Requisitos Não Funcionais

    • RNF01 - Interface responsiva (mobile/desktop)
    • RNF02 - Código modular e documentado
    • RNF03 - Persistência local (LocalStorage)
    • RNF04 - Conformidade com LGPD
    • RNF05 - Processamento assíncrono de notificações
    • RNF06 - Cache de relatórios para performance
    • RNF07 - IA baseada em regras e padrões

    🤖 Sistema IA
    Processar Linguagem Analisar Intenção Gerar Respostas 
    Paciente → Interface: Solicita agendamento
Interface → Controlador: agendarConsulta(dados)
Controlador → Validador: validarDados(dados)
Validador → Controlador: dados válidos
Controlador → DataService: salvarConsulta(consulta)
DataService → Controlador: confirmação
Controlador → NotificationService: agendarLembrete()
Controlador → Interface: atualizarView()
Interface → Paciente: exibir confirmação

    📝 Changelog - Versão 2.0

    ✨ Novas Funcionalidades

    • Chatbot IA - Assistente virtual para atendimento automatizado
